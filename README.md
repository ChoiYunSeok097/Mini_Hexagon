# Mini_Hexagon
Shader를 이용한 게임

# 설명
- 기간:
- 인원: 1인
- 언어: 자바스크립트와 HTML
- 설명: 리듬게임 헥사곤처럼 맵 밖에서 다가오는 장애물을 피하는 게임, 
      자바 스크립트에서 벡터와 색, 계산을 맡고
      HTML에서 vertex-shader와 버퍼를 이용해서 부드러운 영상을 출력합니다.
